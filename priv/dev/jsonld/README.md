This directory holds JSON‑LD examples for local development only.

- Scope: dev-only. Do not publish with any library packages.
- Usage: Paste into the JSON‑LD Runner (/dev/jsonld) or click Copy in the UI.
- Safety: These examples are inert; they exercise dev-only, whitelisted actions.

When extracting the DevKit as a library, exclude this path from the package (e.g.,
package: [files: ["lib", "mix.exs", "README*"], ], or add a package exclude).
